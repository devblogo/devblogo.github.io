栗子前端技术周刊第26期 - pnpm v90、Next

发表于 2024-04-22

🌰栗子前端技术周刊第 26 期 (2024.4.15 - 2024.4.21)：浏览前端一周最新消息，学习国内外优秀文章视频，让我们保持对前端的好奇心。

📰 技术资讯

  1. pnpm v9.0：pnpm 迎来了 v9.0.0 版本，带来了众多改进，包括放弃对 Node.js v16 的支持、Corepack 兼容性、配置更新、改进锁文件、改进依赖关系解析等等。
  2. Next.js 14.2：Next.js 即将迎来八岁生日，月活跃开发者已突破 100 万，此次发布的版本支持使用 Turbopack 改进本地开发、内存使用、CSS 和缓存优化等。
  3. React 18.3 即将发布：React 18.3 即将发布，作为 React 19 发布前最后一个小版本。

📒 技术文章

  1. Building a CLI from Scratch with TypeScript and oclif：使用 TypeScript 和 oclif 从零开始构建 CLI - oclif 是由 Salesforce 维护的成熟 CLI 工具开发框架。本教程将介绍如何从零开始创建可运行的 CLI。
  2. useTransition：React 团队在其官方文档中添加了新的 useTransition 指南
    ，useTransition 是一个 React Hook，可让你在不阻塞用户界面的情况下更新状态。
  3. JSBridge原理 - 前端H5与客户端Native交互：在混合应用开发中，需要解决 H5 与 Native 之间的双向通信，而 JSBridge 就是一种在混合应用中实现 Web 和原生代码之间通信的重要机制。

🔧 开发工具

  1. Quill 2.0：功能强大的 Web 富文本编辑器。在 Quill 2.0 发布中，Quill 完成向 TypeScript 的过渡以及对现代浏览器功能的改进，除此之外还有更多内容，例如其 ESM 打包。


2. TresJS：使用 Vue.js 构建 3D 场景 - 使用 Vue 组件和 Three.js 创建 3D 场景。


3. Wedges：适用于 React 的 UI 组件集 - 由 Lemon Squeezy 制作和使用，这是一套经过深思熟虑、美观大方的基于 Radix UI 和 Tailwind CSS 的组件集。您还可以下载其中的 Figma 文件，以便在模拟布局时使用。
