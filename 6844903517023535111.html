Java 设计模式之模板方法模式（十三）

发表于 2017-11-28

一、前言

上篇《Java 设计模式之代理模式（十二）》 为 Java 设计模式中结构型模式的最后一章，今天开始介绍 Java 设计模式中的行为型模式的第一种模式–模板方法模式。

二、简单介绍

# 2.1 定义

模板方法（ Template Method）模式是行为模式之一，它把具有特定步骤算法中的某些必要的处理委让给抽象方法，通过子类继承对抽象方法的不同实现改变整个算法的行为。

# 2.2 应用场景

  1. 具有统一的操作步骤或操作过程。
  2. 具有不同的操作细节。
