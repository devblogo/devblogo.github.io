SwiftUI 学习笔记【02】：新建SwifUI项目

发表于 2024-04-22

在本文中，你将学会如何使用xCode创建一个SwifUI项目

1 首先，启动 xCode 之后，根据当前是否已经打开已有项目，分两种情况来创建 SwiftUI 项目，
1.1 若没有打开已有项目，显示如下界面的话，则点击“Create New Project…”菜单项

1.2 若已打开已有项目，在顶栏创建一个新项目，选择File，New，Project

2 接着选择 iOS 页签，再选择 Application 下的 App，最后点击“Next”按钮

3 完善要创建的项目的基本信息


在上面的截图中，完善大红框里面的内容，

  • 项目名称(Project Name)：FirstApp；
  • 组织名称(Team)：可以设置为“公司”或“个人组织”，也可以选择 None
  • 组织标识符(Organiztion Identifier)：一般使用反向的域名，例如：com.exampe；
  • 反向域名(Bundle Identifier)：自动生成，组织标识符+项目名称；
  • 用户界面(Interface)：选择SwiftUI；
  • 语言(Language)：选择Swift；
  • 语言(Storage)：选择None；

完善项目信息后，点击“Next”按钮

4 选择保存目录


在这一步选择要保存的目录，若勾选了“Create Git repository on my Mac”，xCode将为该项目初始化 Git 仓库，至此你就成功创建了一个SwiftUI项目。
